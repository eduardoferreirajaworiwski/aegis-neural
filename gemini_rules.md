# Aegis-Neural DNA

## Coding Agency
- **Modular Structure:** Maintain a clean, modular structure. Use separate files for model definitions, data preprocessing, and training loops.
- **Tool Usage:** Leverage `write_file`, `read_file`, and `run_shell_command` to manage the codebase effectively.

## Tech Stack Mastery
- **Neural Networks:** Enforce **PyTorch** for all neural network implementations.
- **NLP/Preprocessing:** Use **Scikit-learn** for data preprocessing and NLP tasks.

## Safety & Verification
- **Validation Step:** Every training script must include a validation step (test split) to prevent overfitting.
- **Data Integrity:** Validate input data shapes and types before processing.

## Explainability
- **Shape Transformation:** For every Neural Network layer created, explain the Shape Transformation (Input Tensor vs. Output Tensor) in comments or documentation.
